# Helix (React rebuild)

A student portal — dashboard, weekly timetable, folder-based study materials, announcements,
profile management, and an AI assistant — rebuilt from the original single-file HTML/JS version
into a proper React + Vite app.

## Stack

- **Frontend:** React 18 + Vite + Tailwind CSS (real build, not the CDN script)
- **Auth & database:** Firebase (Authentication + Firestore)
- **File storage:** Supabase Storage
- **AI:** Google Gemini (`gemini-3.5-flash`)
